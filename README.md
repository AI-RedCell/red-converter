# RED CONVERTER

> Advanced Text Transformation Suite by AI RedCell

## Overview

RED CONVERTER is a professional-grade, client-side text transformation and steganography toolkit. It's designed as a Progressive Web App (PWA) with offline capabilities, featuring 40+ encoding, cipher, and steganography transformations.

## Features

- **40+ Transformation Algorithms** - Base encodings, ciphers, steganography, and obfuscation
- **Visual Pipeline Builder** - Chain multiple transformations together
- **Auto-Detection** - Automatically identify encoding types
- **100% Client-Side** - All processing happens locally, no data sent to servers
- **PWA / Offline Ready** - Works without internet connection
- **History & Favorites** - Save and revisit past transformations

## Technologies

- **React 18** + **TypeScript**
- **Vite** (build tool)
- **Tailwind CSS** + **shadcn/ui**
- **Framer Motion** (animations)
- **IndexedDB** (local storage)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
